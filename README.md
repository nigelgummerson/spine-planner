**Leeds Spinal Planner** is a digital whiteboard designed for spinal surgeons to plan and document correction maneuvers. It replaces manual paper sketching with a clean, exportable digital chart (PDF/JPEG) suitable for operation notes and scrubbing teams.

**No Install Required:** Runs as a single HTML file in any browser.

**Deformity Focused:** Dedicated tools for Dual Differential Bending (DDB) forces, osteotomies, and vertebral column resection.

**Privacy First:** "Session Mode" ensures no patient data is retained on shared clinical workstations.
